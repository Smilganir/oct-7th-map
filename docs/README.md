# Tableau dashboard (GitHub Pages)

This folder contains a single-page responsive layout that embeds the **Oct 7th Otef** Tableau Public dashboard.

## Publish on GitHub Pages

1. **Push this repo to GitHub** (if you haven’t already), including the `docs/` folder and `docs/index.html`.

2. **Turn on GitHub Pages:**
   - Open your repo on GitHub → **Settings** → **Pages**.
   - Under **Build and deployment** → **Source**, choose **Deploy from a branch**.
   - **Branch:** `main` (or your default branch).
   - **Folder:** `/docs`.
   - Click **Save**.

3. After a minute or two, the site will be at:
   - **Project site:** `https://<username>.github.io/<repo-name>/`
   - **User site:** `https://<username>.github.io/` (if the repo is named `<username>.github.io`).

The Tableau dashboard is embedded in `index.html` and will load from Tableau Public when visitors open the page.
